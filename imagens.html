Estudando HTML

Nossos estudos começaram falando a respeito de um item muito sério que é a organização. Isso porque a linguagem html é responsável apenas pela marcação dos elementos na janela do navegador. O html não cria nada, ele manda buscar ou utilizar as letrinhas que você digita.

Parágrafos

Para colocarmos parágrafos, usamos elementos "p" e "/p". Para fazer quebra de linha, mantendo o mesmo parágrafo, usamos o elementos "br". Temos ainda parágrafos especiasis, denominados títulos, cabeçalhosou headings em que são usados elementos de "h1" a "h6" e seus respectivos fechamentos de "/h1" a "/h6".

Imagens

Uma imagem aparece na janela do navegador pelo uso do elemento "img". Esse elemento possui atributo "src", responsável pela localização da imagem. Para trabalhar melhor com a imagem, você pode colocá-la dentro das tags de marcação de parágrafo.
